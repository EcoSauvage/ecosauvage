ÉcoSauvage

Ensemble autonome

Orientation

ÉcoSauvage est un lieu autonome. Pas de visites, pas de tourisme, pas d’événementiel, pas de commercial. Le lieu avance par cohérence, contributions et excédents. Lecture avant contact recommandée.

Gradients d'entrée

Progression volontaire : curieux compatibles → soutiens → actifs → collaborateurs → membres. Ni ouvert à tous, ni fermé ; question de pertinence, utilité, autonomie.

Contributions

Possibles à distance (matériel, financier, compétences) ou sur place (selon besoins). Objectif : autonomie + excédents.

Cadre opérationnel

Arbitrages techniques : eau → énergie → serre → poules → bois → déchets → safran → excédents. Safran interne. Excédents pour soutenir l’ensemble.

Première prise de contact

Pour un premier message : se situer, indiquer ce qui est proposé (matériel, financier, compétences, disponibilité) et l’horizon possible. Le lieu répond lorsqu’il y a alignement.
